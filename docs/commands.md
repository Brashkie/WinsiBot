# Comandos de WinsiBot

Los comandos se activan con cualquiera de los prefijos configurados (`!`, `.`, `#`, `/`).

---

## Admin

| Comando | Descripción | Permiso |
|---------|-------------|---------|
| `!ban @usuario` | Expulsar y banear del grupo | Admin |
| `!kick @usuario` | Expulsar del grupo | Admin |
| `!promote @usuario` | Promover a administrador | Admin |
| `!demote @usuario` | Quitar administrador | Admin |
| `!antilink on/off` | Activar/desactivar anti-enlaces | Admin |
| `!mute` | Silenciar el bot en el grupo | Admin |
| `!unmute` | Reactivar el bot | Admin |

## Inteligencia Artificial

| Comando | Descripción |
|---------|-------------|
| `!ai <consulta>` | Chat con IA |
| `!gpt <consulta>` | Respuesta GPT |
| `!imagine <descripción>` | Generar imagen con IA |

## Descargas

| Comando | Descripción |
|---------|-------------|
| `!yt <url o nombre>` | Descargar de YouTube (audio/video) |
| `!tiktok <url>` | Descargar video de TikTok |
| `!ig <url>` | Descargar de Instagram |
| `!fb <url>` | Descargar de Facebook |

## Stickers

| Comando | Descripción |
|---------|-------------|
| `!sticker` | Convertir imagen/video a sticker |
| `!toimg` | Convertir sticker a imagen |
| `!emojimix <emoji+emoji>` | Mezclar dos emojis |

## Música

| Comando | Descripción |
|---------|-------------|
| `!play <nombre>` | Buscar y enviar canción |
| `!lyrics <nombre>` | Letra de canción |

## Información

| Comando | Descripción |
|---------|-------------|
| `!owner` | Información del dueño |
| `!groups` | Lista de grupos del bot |
| `!uptime` | Tiempo activo del bot |
| `!ping` | Latencia del bot |

## Sub-bots (JadiBot)

| Comando | Descripción |
|---------|-------------|
| `!jadibot <número>` | Vincular nuevo sub-bot |
| `!stopbot <número>` | Detener sub-bot |
| `!listbot` | Ver sub-bots activos |
